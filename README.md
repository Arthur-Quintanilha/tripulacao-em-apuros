# Tripulação em Apuros

Jogo de furtividade e sobrevivência ambientado em um navio pirata. Sequência de abertura: Porão → Convés.

## Como jogar

Abra `index.html` no navegador (ou use um servidor local).

```bash
npx serve .
```

### Controles

| Ação | Tecla/Botão |
|---|---|
| Mover | WASD / Setas |
| Pular (subir na caixa) | Espaço |
| Agachar / modo furtivo | Shift |
| Atacar (eliminação furtiva) | Botão esquerdo do mouse |
| Interagir / avançar diálogo | E ou clique |

### Sequência

1. **Porão** — Fale com o marujo caído e receba o canivete
2. **Porão** — Pule na caixa para alcançar a escada quebrada e sair
3. **Convés** — Aproxime-se furtivamente (Shift) do pirata na sala do capitão e ataque com clique
4. **Convés** — Pegue as chaves e fuja agachado até o bote salva-vidas

## Tecnologias

- HTML5, CSS3, JavaScript
